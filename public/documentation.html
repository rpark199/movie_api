Movie API

Movieflix is built with the REST architeture. The API will provide users with access to information about movies. Users are able to sign up, update their personal information, and create a list of their favorite movie.

Requests

Request URL HTTP Method Parameters Request body data format Response body data format
Return a list a all movies to the user /movies GET None None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[Title] Get :title None A JSON object holding all the data about a single movie. Example:
{ 'title': 'The Dark Knight',
'director': 'Christopher Nolan',
'genre': 'action'}
Return data about a genre by name /movies/genre/[genreName] GET :genre None A JSON object holding data about movies belonging to a specific genre. For Example:
{ title: 'The Terminator', director: 'James Cameron', genre: 'Science fiction action' }
{ title: 'The Shawshank Redemption', director: 'Frank Darabont', genre: 'action thriller' }
Return data about a director by name /movies/directors/[directorName] GET :director None A JSON object holding data about a single director, containing director name. Example:
{ name: 'Christopher Nolan', bio: 'A short Bio', dob: '30th July' }
Allow new users to register /users/register POST :register Data containing new user details A text response indicating that the registeration was successful:
Allow users to update their user info by Username /users/update/[id]] PUT :id Data containing changes to old profile A text response indicating that the changes were successfully deployed
Allow users to add a movie to their list of favorites /favorite/add/[id]/[title] POST :add A JSON object holding data about the new data about the movie to be added. For example:
{ title: 'Wrath Of Man', director: 'Guy Ritchie', genre: 'Action-Thriller' } 		A JSON format data about the newly added movie details.
{ "id": 1, "fullname": "John Doe", "email": "johndoe@mail.com", "favMovies": [ { "title": "Inception", "director": "Christopher Nolan", "genre": "Sci-Fi" }, { "title": "Wrath Of Man", "director": "Guy Ritchie", "genre": "Action-Thriller" } ] }
Allow users to remove a movie from their list of favorites /favorite/delete/[id]/[title] DELETE :id, :title None An array without the removed movie
Allow existing users to deregister /users/deregister/[id] DELETE :id None A text message indicating whether the user deregister successfully
